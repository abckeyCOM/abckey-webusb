## abckey-webusb
[![NPM version](https://img.shields.io/npm/v/@abckey/webusb.svg)](https://www.npmjs.com/package/@abckey/webusb)

Communicate with ABCKEY and Trezor Hardware Wallets.

- Already support trezor one.

## Install

```
npm i @abckey/webusb
```
